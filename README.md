# team_building_server
team building server

## 团队建设系统前端服务

## 下载项目、安装modules
- `git clone`
  ```
  git clone https://github.com/codingview/team_building_server.git
  ```
- 安装`modules`
  ```
  npm install 
  ```

## 安装&启动项目

- 安装`eslint`
  ```
  npm install -g eslint
  ```
- 安装`google-eslint`规则
  ```
  npm install -g eslint-config-google
  ```
- `WebStorm`配置`eslint` ......
